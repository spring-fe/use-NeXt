## 介绍
NeXt是一个基于HTML5/Javascript框架，提供了一个具有高性能和丰富功能的拓扑图UI组件。NeXt可以与DLUX一起构建ODL应用程序。
#### 主要特征
- 支持大型的复杂的网络拓扑
- 支持聚合的网络节点
- 流量/路径/隧道/组可视化
- 拥有不同的拓扑图布局算法
- 可以以地图为背景层展示拓扑图
- 友好的用户交互
